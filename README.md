# CS-210
Programming Languages: SNHU 2025

CS 210 built my understanding of how programming languages are structured differently — not just in syntax, but in the way they model problems. Python's Hello World took one line; C++ required #include, a main() function, and explicit namespace management. That contrast revealed how much work a language does silently on the programmer's behalf, and how understanding what is happening under the hood changes how you write code.

The two projects reinforced the value of object-oriented design. In Project 2, separating the InvestmentCalculator class from main() meant that the calculation logic could be written, read, and tested independently from the user interface. The header file made the class's public interface explicit before a single line of implementation was written — a discipline that makes larger programs much easier to manage.
